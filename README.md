# 💳 Transaction Fraud Detection System 

## 📌 Project Overview

This project is a Machine Learning-based Fraud Detection System that identifies whether a financial transaction is **fraudulent or legitimate**.

The model is trained on transaction data and uses various data science techniques to detect suspicious activities.

---

## 🚀 Features

* Detect fraudulent transactions
* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Visualization using graphs
* Machine Learning model training and evaluation

---

## 🛠️ Tech Stack

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📊 Dataset

The dataset used in this project contains transaction details such as:

* Transaction amount
* Sender and receiver information
* Transaction type
* Fraud label (0 = Legit, 1 = Fraud)

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation

---

## 📈 Model Used

* Logistic Regression
* Random Forest

---

## 📌 Results

The model successfully identifies fraudulent transactions with good accuracy and helps in reducing financial risks.
